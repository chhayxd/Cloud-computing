Enhancements

Enhancement 1 - Responsive Layout

This website includes a responsive layout using CSS media queries. The navigation, content cards, and topic page layout automatically adjust when the screen becomes smaller. This goes beyond the basic requirements because it improves usability on tablets and mobile devices.

The code is implemented in the stylesheet using @media rules that change the flex layout into a stacked layout on smaller screens.

It is applied on the Home page and Topic page.

Enhancement 2 - Hover Effects on Navigation and Cards

This website also uses hover effects on the menu and homepage cards. These effects improve interactivity and help guide the user to clickable content. This is an enhancement because it adds a more polished and engaging user experience than a plain static layout.

The code uses CSS pseudo-classes such as :hover to change colours, movement, and shadows when the user moves the mouse over elements.

It is applied on the Home page and all navigation menus.
